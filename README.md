# E-Store App

This is an E-Store where users explore, search, filter, add product/s into the cart and check out.

## Used Tools and Technologies

<b>Front-End</b>

- Vite
- React
- Redux
- TypeScript
- Axios
- Material UI

<b>Back-End</b>

- .Net 7 (API)
- Entity Framework
- SQLite (Devlopment)
- PostgreSQL (Production)
